# 🌿 MindSpark Telegram Bot

Умный Telegram-бот для управления репетиторским сервисом MindSpark с ИИ-ассистентом.

## ✨ Возможности

### 📚 Основной функционал
- 📖 Запись на уроки с выбором даты и времени
- ✨ Пробные уроки для новых учеников
- 🌿 Личный кабинет с прогрессом и материалами
- 🪙 Система оплаты с подтверждением чеков
- 📜 Система отзывов с модерацией
- 💡 ИИ-ассистент на базе Groq Llama 3.3

### 🤖 ИИ-ассистент
- Модель: `llama-3.3-70b-versatile`
- Быстрые ответы через Groq API
- Помощь с учёбой на русском языке
- Обработка ошибок и логирование

### 📅 Система записи
- Конкретные даты вместо дней недели
- Валидация формата ДД.ММ.ГГГГ
- Автоматические напоминания за час до урока
- Фоновая проверка каждые 60 секунд

### 🛠 Админ-панель
- 👥 Управление учениками
- 📅 Просмотр всех записей
- 💰 Статистика оплат
- ✏️ Управление домашними заданиями
- 📢 Умная рассылка с фильтрами
- 📢 Общая рассылка

## 🚀 Установка и запуск

### Локальный запуск
1. Клонируйте репозиторий
2. Установите зависимости:
```bash
pip install -r requirements.txt
```
3. Создайте файл `welcome.png` для приветственного изображения
4. Запустите бота:
```bash
python study_attic.py
```

### Переменные окружения
- `BOT_TOKEN` - токен Telegram бота
- `GROQ_API_KEY` - API ключ для Groq
- `ADMIN_ID` - ID администратора

## 📦 Зависимости

```
aiogram==3.7.0
groq
```

## 🗄 База данных

### Таблицы
- `students` - информация об учениках
- `bookings` - записи на уроки
- `payments` - платежи
- `homework` - домашние задания
- `homework_submissions` - сданные ДЗ
- `reviews` - отзывы
- `materials` - учебные материалы

## 🎮 Команды

### Пользовательские команды
- `/start` - запуск бота с приветствием
- `/admin` - админ-панель (только для админа)

### Админские команды
- `/approve [review_id]` - одобрить отзыв
- `/confirm [payment_id] [user_id]` - подтвердить оплату
- `/add_material [user_id] [title] [link]` - добавить материал
- `/set_progress [user_id] [%] [уроков]` - обновить прогресс
